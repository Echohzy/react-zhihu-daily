# react-zhihu-daily

React版知乎日报，包含热门文章，最新文章以及主题，主要是想尝试一下Typescript。

## Usage

```
git clone https://github.com/Echohzy/react-zhihu-daily.git
cd react-zhihu-daily
npm install
npm run dev
```

然后打开浏览器，访问http://localhost:3000，即可看到结果。
